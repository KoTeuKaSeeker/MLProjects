Проект: PredictNextSymbolRNN#1
Описание:
В этом проекте реализованна простейшая реккурентная нейронная сеть, которая принимает на вход последовательность из нескольких символов, а на выход выдаёт следующий символ, который вероятнее всего следовал бы после входной последовательности.

При запуске программы сеть обучается на тексте, который находится в файле text.txt. Затем пользователю предлагается ввести некую последовательность символов, которая затем продолжится с помощью реккурентной нейронной сети.
