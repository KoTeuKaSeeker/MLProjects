Проект: PredictNextWordRNN#2
Описание:
В этом проекте реализованна простейшая реккурентная нейронная сеть, которая принимает на вход последовательность из нескольких слов, а на выход выдаёт следующее слово, которое вероятнее всего следовало бы после входной последовательности.

При запуске программы сеть обучается на тексте, который находится в файле text.txt. Затем пользователю предлагается ввести некую последовательность слов, которая затем продолжится с помощью реккурентной нейронной сети.
